# Mein Portfolio - Marco

## Beschreibung

Dies ist meine erste einfache Portfolio-Webseite.
Sie enthält eine kurze Vorstellung meiner Person,
eine Übersicht meiner Skills und ein Kontaktformular.

## Verwendete Techniken

- HTML5 (semantische Struktur mit header, main, section, footer)
- CSS Styling
- Flexbox für Navigation und Formularlayout
- Hover Effekte
- Responsive Design mit Media Queries

## Weg

Ich habe Weg B (Portfolio-Webseite) gewählt.

## Notizen

Die Navigation nutzt interne Links zu den einzelnen Sections.
Das Layout wurde mit Flexbox umgesetzt und für kleinere Bildschirme
durch eine Media Query angepasst.
